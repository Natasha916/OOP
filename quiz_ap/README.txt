# Quiz Application Setup Guide

## 1. Install Required Software
- Download and install Java (JDK 8 or later).
- Download and install MySQL Server & MySQL Workbench.
- Install Eclipse or IntelliJ (recommended for Java development).

## 2. Setup the Database
- Open MySQL Workbench and connect to your database server.
- Run the `setup_database.sql` file to create the required tables.

## 3. Run the Application
1. Open Eclipse or IntelliJ.
2. Create a new Java project and copy all `.java` files into the `src/competition` directory.
3. Ensure the MySQL JDBC Driver (`mysql-connector-java.jar`) is added to your project.
4. Run `Main.java` to start the application.

## 4. Features
- User Signup/Login
- Play Quiz with Beginner, Intermediate, and Advanced levels
- Admin can add/edit/delete questions
- View high scores
